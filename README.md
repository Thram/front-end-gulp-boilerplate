# Front-End Gulp Boilerplate
A nice Gulp Boilerplate for Front-End Development

## Includes:
- Browserify
- Sass
- Browser Sync
- Source Maps for JS & CSS on Debug Mode
- JS Lint
- Optimize Images
- Run Tests with Mocha
- Include Partials (@@include) or Templates (%%include)

## Build Commands
### Debug Mode
```javascript
gulp
```
### Release Mode
```javascript
gulp -r // gulp --release
```
